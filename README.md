FECS

### 安装

```
    $ [sudo] npm install fecs -g
```

### 使用

```
    fecs
    fecs -v
    fecs check --help
    fecs format --help
```

更多参数见 wiki: [CLI](https://github.com/ecomfe/fecs/wiki/CLI)
