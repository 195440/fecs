FECS

基于Node.js的前端代码风格工具

  • 灵活

    基于 Nodejs
    安装配置极其容易

  • 高效

    只需一个命令
    解决所有问题

  • 齐全

    包含对 JS、CSS、HTML
    全方位的检查

基于四个工具

  • htmlcs

    检查、 格式化
    html 代码规范

  • csshint

    检查 CSS 代码规范

  • lesslint

    检查 LESS 代码规范

  • jsformatter

    修复 js 代码

安装 
$ [sudo] npm install fecs -g
使用 
fecs
fecs -v
fecs check --help
fecs format --help

开源的ETpl来自百度EFE的RIA团队。在 EFE，我们还有其他开源产品：

  • ECharts直观，生动，可交互，可高度个性化定制的数据可视化图表。
  • ESL比 Require.js 更精简、更高效、更健壮的 AMD Loader。
  • EDP提供了前端开发时常用工具：项目管理、包管理、调试、构建、代码生成、代码检测等，并允许用户自定义扩展。
  • EST基于 Less 的样式工具库，提供了一系列方便快捷的 mixin，只在调用时才输出代码。
  • Saber为移动端与现代浏览器设计的，细粒度、模块化的前端应用框架。
  • Rebas与 Saber 搭配使用、基于 Node 的 JavaScript 同构解决方案。
  • Rider基于 Stylus 与后处理器，无侵入风格的 CSS 样式工具库。
  • ER富浏览器端 Web 应用的框架，适用于并能很方便地构建一个整站式的 AJAX Web 应用。
  • ESUI一套功能完善，并具有强大扩展机制的 UI 组件库。基于模板、声明式地编程方式让视图更具表达能力。
  • ETpl强复用、灵活、高性能的 JavaScript 模板引擎。
  • iSlider表现出众，无任何插件依赖的手机平台 JavaScript 滑动组件。
  • 图说一个向广大用户提供数据可视化服务平台。
  • Fontmin首个纯 JavaScript 字体子集化方案

......
